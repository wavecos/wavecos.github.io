Instalando Docker en Ubuntu 14.04

February 13, 2016

¿ Qué es Docker?

Docker es un proyecto open source que automatiza el despliege de aplicaciones dentro de contenedores de software. mas información aquí.

Instalando Docker en Ubuntu.

Lo primero es actualizar todos los paquetes de Ubuntu antes de instalar algo.

sudo apt-get update

Ahora instalemos docker con su respectivo paquete:

sudo apt-get install docker.io

Linkeamos rutas

sudo ln -sf /usr/bin/docker.io /usr/local/bin/docker
